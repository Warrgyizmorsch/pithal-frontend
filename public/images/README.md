# Image Replacement Guide

The SVG files in this directory are branded placeholder media used to keep the homepage runnable and responsive before final Figma exports arrive.

Replace images in place, retaining these paths where possible:

- `hero/hero-machine-photo.png`, `hero/engineered-machine.svg`, `hero/video-showcase.svg`
- `products/*.svg`
- `industries/*.svg`
- `projects/*.svg`
- `process/*.svg`

Optimized `.webp` or `.avif` photography can also be used by updating the corresponding entries in `src/data/homeData.ts` and direct section image paths.
